Ejercicio #1: Cálculo del IMC

El IMC se usa como una herramienta de detección, pero no diagnostica la grasa corporal ni la salud de un individuo. Para determinar si el exceso de peso es un riesgo para la salud, un proveedor de atención médica necesitará realizar evaluaciones adicionales. Para calcular el IMC se necesita conocer el peso en kilogramos y la estatura en metros. Se aplica una sencilla fórmula matemática que consiste en dividir el peso entre la estatura al cuadrado: IMC = Peso (Kg) / Estatura al cuadrado (Mt)
